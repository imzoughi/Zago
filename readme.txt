Paris&CO

Build & development

- Download the last version of node
- cd ParisCo/
- npm install
- npm install -g bower
- bower install (install front dependencies inside a directory called "inc")
- Do "grunt" for run tasks by default
- to see the HTML/CSS development, 3 files : index.html
- There is a task "grunt watch"



Production
- Do "grunt deployProd", It's the same command by default "grunt" but with the javascript minified.

For each components css, there is a file inside the "component" directory.
